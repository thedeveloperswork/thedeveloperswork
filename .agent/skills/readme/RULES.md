# Rules for the GitHub Profile README

When modifying the `README.md` file, strictly adhere to the following rules:

1. **Alignment:** Use `<div align="center">` or `<h4 align="center">` for main headers, badges, and stats cards to maintain the centered aesthetic.
2. **Snake Animation:** The contribution snake animation must strictly use this source URL: `https://raw.githubusercontent.com/TheDevelopersWork/developerswork/output/snake.svg`.
3. **GIF Sizing:** Any decorative GIFs (e.g., from media.giphy.com) should have a `width="30"` for inline text, or an appropriate max width/height to prevent layout shifting.
4. **Widgets & Stats Card:** The profile uses `github-readme-activity-graph.vercel.app`, `github-readme-streak-stats.herokuapp.com`, and `gh-trophy.cdnsoft.net`. Do not remove or alter these widgets' themes (e.g. `theme=radical` or `theme=aura`) unless explicitly requested.
5. **Social Links:** The "Let's Connect!" section must use `img.shields.io` badges for all social links, consistently styled with `style=for-the-badge`. Keep them strictly categorized as defined in `assets/05_social.md`.
6. **Spacing:** Maintain clean vertical spacing between `<div align="center">` blocks using horizontal rules `---` or `<br/>` where appropriate to avoid visual clutter.
